# PHP Calculator

This is a simple calculator project built for PHP fresher interview practice.

## Technologies Used
- HTML
- CSS
- Bootstrap
- PHP

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Divide by zero handling
- Same page result display

## How to Run
1. Install XAMPP
2. Copy project folder into `htdocs`
3. Start Apache
4. Open browser and run:
   http://localhost/yahoo.php/calculator.php
